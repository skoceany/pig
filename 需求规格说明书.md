# 智能养殖猪病诊断智能体 - 需求规格说明书

## 1. 引言

### 1.1 编写目的
本文档详细描述基于大模型的智能养殖智能体的功能需求、非功能需求、数据需求等，为系统开发提供明确的指导和依据。

### 1.2 项目背景
随着规模化养殖的发展，猪病诊断成为养殖产业的关键环节。传统的人工诊断方式效率低、主观性强，且难以覆盖大规模养殖场的实时监测需求。本项目旨在利用大模型和多模态数据技术，构建一个智能猪病诊断系统，实现自动化、智能化的猪病识别与分析。

### 1.3 术语定义
| 术语 | 定义 |
|------|------|
| 智能体 | 基于大模型的AI应用，能够接收输入并产生智能响应 |
| 多模态数据 | 包含图像、文本等多种类型的数据 |
| 微调 | 在预训练模型基础上，使用特定领域数据进行训练 |
| 提示词工程 | 设计合适的输入提示词以引导模型产生期望输出 |

## 2. 功能需求

### 2.1 核心功能

#### 2.1.1 图像诊断功能
- **功能描述**：用户上传一张猪场中猪的照片，系统对照片进行分析，判断猪是否患病
- **输入**：猪的照片（支持JPG、PNG格式）
- **输出**：
  - 是否患病的判断结果（患病/健康）
  - 若患病，给出具体的疾病名称
  - 对疾病的详细解读（症状描述、病因、传播途径等）

#### 2.1.2 疾病知识库查询
- **功能描述**：系统内置12种常见猪病的知识库，支持根据疾病名称查询详细信息
- **支持的疾病类型**：
  1. 非洲猪瘟
  2. 猪瘟
  3. 猪口蹄疫
  4. 猪繁殖与呼吸综合征
  5. 猪圆环病毒病
  6. 猪传染性胃肠炎
  7. 猪伪狂犬病
  8. 猪链球菌病
  9. 副猪嗜血杆菌病
  10. 猪丹毒
  11. 猪传染性胸膜肺炎
  12. 猪附红细胞体病

### 2.2 辅助功能

#### 2.2.1 诊断历史记录
- **功能描述**：记录用户的诊断请求历史，便于查看和管理
- **记录内容**：诊断时间、上传图片、诊断结果、疾病解读

#### 2.2.2 诊断结果分享
- **功能描述**：支持将诊断结果分享给其他用户或保存为本地文件

## 3. 非功能需求

### 3.1 性能需求
- **响应时间**：从上传图片到返回诊断结果的时间不超过10秒
- **并发处理**：支持至少10个用户同时进行诊断请求

### 3.2 可靠性需求
- **诊断准确率**：疾病识别准确率不低于85%
- **系统可用性**：系统正常运行时间不低于99%

### 3.3 兼容性需求
- **平台支持**：支持Windows、Linux操作系统
- **浏览器支持**：支持Chrome、Firefox、Edge等主流浏览器
- **图片格式**：支持JPG、PNG、BMP等常见图片格式

### 3.4 安全性需求
- **数据加密**：用户上传的图片和诊断结果进行加密存储
- **访问控制**：系统提供用户认证机制，确保数据安全

## 4. 数据需求

### 4.1 训练数据
- **数据来源**：多模态数据/train-data目录
- **数据结构**：
  - 12个子目录，每个子目录对应一种疾病
  - 每个子目录包含：
    - 疾病相关图片（JPG格式）
    - 描述.txt文件（包含疾病概述和每张图片的详细描述）
- **数据规模**：共352张图片，12种疾病

### 4.2 数据处理要求
- **图像预处理**：统一图片尺寸、格式转换、数据增强
- **文本处理**：提取描述文件中的疾病知识，构建结构化知识库

## 5. 系统架构

### 5.1 整体架构
系统采用分层架构，包括：
1. **前端展示层**：用户交互界面，负责图片上传和结果展示
2. **业务逻辑层**：处理诊断请求，调用AI模型进行分析
3. **AI模型层**：基于大模型的图像分析和疾病诊断模型
4. **数据存储层**：存储训练数据、诊断历史和知识库

### 5.2 技术选型
- **前端框架**：Vue.js 3
- **后端框架**：FastAPI（Python）
- **AI模型**：基于LLaVA或Qwen-VL等多模态大模型
- **数据库**：SQLite（轻量级，适合小规模应用）

## 6. 接口需求

### 6.1 API接口

#### 6.1.1 图像诊断接口
- **接口地址**：/api/diagnosis
- **请求方法**：POST
- **请求参数**：
  - image: 图片文件（multipart/form-data）
- **响应格式**：
```json
{
  "success": true,
  "data": {
    "is_sick": true,
    "disease_name": "非洲猪瘟",
    "confidence": 0.92,
    "interpretation": "非洲猪瘟的潜伏期因病毒株和猪的品种而异...",
    "symptoms": ["高烧", "皮肤紫斑", "内脏出血"],
    "recommendations": ["立即隔离", "通知兽医", "消毒处理"]
  }
}
```

#### 6.1.2 疾病知识库查询接口
- **接口地址**：/api/disease/{disease_name}
- **请求方法**：GET
- **响应格式**：
```json
{
  "success": true,
  "data": {
    "disease_name": "非洲猪瘟",
    "description": "非洲猪瘟是由非洲猪瘟病毒引起的急性传染病...",
    "symptoms": ["高烧", "皮肤紫斑", "内脏出血"],
    "transmission": "接触传播",
    "treatment": "尚无特效药物"
  }
}
```
